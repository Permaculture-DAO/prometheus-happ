# Publication Boundary

This repository is publicly visible to support verification of the signed runtime
proof and bounded technical review.

Public visibility does not:

- make the repository open source while its packages remain `UNLICENSED`;
- create production, security, performance, empirical, legal, market, or
  financial certification;
- authorize real-world evidence ingestion or capital-facing use;
- convert synthetic TEST evidence into ecological proof or admissible value;
- alter the authority of the signed canon in `prometheus-canon`.

The release claim is limited to the evidence package and known limitations
distributed with `v1.1.3-runtime-proof`.
